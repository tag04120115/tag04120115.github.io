분석 결과 + 16쪽 템플릿으로 구현시키기

방금 Product Design으로 분석한 레퍼런스 명함 홈페이지의 구조와 분위기를 참고해서,

진행 방식:
1. 레퍼런스의 섹션 순서, 여백, 카드 구성, 버튼 느낌을 최대한 참고해줘.
2. 개인정보, 사진, 문구는 반드시 내가 제공한 정보와 첨부 사진으로 교체해줘.
3. 첨부한 사진은 프로필 사진이다. Hero 영역과 프로필 영역에 자연스럽게 사용해줘.
4. GitHub Pages User site에서 바로 실행되는 정적 사이트로 만들어줘.
5. 파일은 index.html, style.css, script.js 중심으로 구성해줘.
6. 모바일에서 먼저 보기 좋게 만들고, PC에서는 넓게 확장되도록 해줘.
7. SEO 기본 태그와 네이버/구글 소유확인 meta 태그를 넣을 주석 위치를 <head> 안에 만들어줘.
